# Exclusive-Pre-Selection
Welcome to Project Titan. This is not a standard Capture The Flag (CTF) event. This is a private, invitation-only technical assessment framework developed by the Google SecRec (Security Recruitment) team.

Our goal is to move beyond traditional resume screening. This repository contains a series of progressively difficult challenges designed to identify candidates who possess not just theoretical knowledge, but the practical, gritty problem-solving skills required for the Security & Engineering Internship.

If you are reading this, you have been specifically identified by our talent scouts or referred by a current Google engineer. Consider this your first interview.
